## Requirements

Project is configurated to use PostgreSQL Database.

You need to create two databases:

<p>movies-api - development database</p>
<p>movies-api-test - tests database</p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
